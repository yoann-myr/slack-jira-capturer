---
title: "Night cooling"
---

Enable **night-time operation** where:

- Airflows are increased at night to accumulate cooling when outside temperature allows it.
- Feature is activated in SOLIDA with a main control.
  - If pressure sent is below 30 Pa then close the AHU
  - If pressure sent is over 30 Pa then start the AHU
- Operate between 00:00 and 06:00 using time schedules / Comfort Schedule
- Supply air temperature setpoints are allowed to go **lower than normal minimum limits**.
- We use a manual comfort schedule that allows pressure to go to 0 Pa.
- We haven’t yet tested the feature that allows temperature to go lower to cool the building.
