title: "Quickstart"
description: "Run the docs locally and deploy via GitHub"
---

## Run locally

<Steps>
  <Step title="Install the CLI">
    ```bash
    npm i -g mint
    ```
    Requires Node.js v20.17.0 or newer.
  </Step>
  <Step title="Start the dev server">
    From the `docs/` folder (where `docs.json` lives):
    ```bash
    mint dev
    ```
    The site is now live at [http://localhost:3000](http://localhost:3000).
  </Step>
  <Step title="Edit a page">
    Change any `.mdx` file and the browser hot-reloads.
  </Step>
</Steps>

## Deploy via GitHub

<Steps>
  <Step title="Push this folder">
    Commit `docs/` to your GitHub repo.
  </Step>
  <Step title="Install the Mintlify GitHub App">
    Go to the [Mintlify dashboard](https://dashboard.mintlify.com) and connect the repo. Point it at the `docs/` folder.
  </Step>
  <Step title="Push to deploy">
    Every push to your production branch rebuilds the site automatically.
  </Step>
</Steps>
